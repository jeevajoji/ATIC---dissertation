 BLOCK 2 — SWIN TRANSFORMER ENCODER

  4 stages, each stage = 2 Swin Transformer blocks
  Each stage halves spatial resolution, doubles channels

  Layer 1: 57,600 tokens → fine texture (bee wings, water ripples) /2
  Layer 2: 14,400 tokens → mid-level structure (shapes, edges)     x2

  Layer 3: 3,600 tokens → objects (boat, horizon, flower)          x2
  Layer 4: 900 tokens → scene semantics (sky, water, macro)        x2

  Inside EACH Swin block — 3-layer attention stack:
  
  ┌─────────────────────────────────────────┐
  │ LAYER A — Shifted Window Attention      │        # kernel or linear patch
  │   Attention within 7×7 local windows    │
  │   Windows shift between layers →        │
  │   local + global context over depth     │
  │                  ↓                      │
  │ LAYER B — Spatial Attention Gate        │        # gating and context
  │   Produces H×W map of values 0→1        │          aware quant map
  │   High value = complex region           │
  │   Low value  = simple region            │
  │   HoneyBee wings → 0.9                  │
  │   Smooth sky     → 0.1                  │
  │                  ↓                      │
  │ LAYER C — Channel Attention (CBAM)      │        # learning whats needed  
  │   Across feature channels               │         
  └─────────────────────────────────────────┘
         │
-

  Output: compact latent tensor 
  (900 tokens × 512 channels)

 BLOCK 3 — ATTENTION-GUIDED ADAPTIVE QUANTIZER
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  Takes: latent z + spatial attention map from Block 2

  Spatial attention map → quantization step size per region:
    High attention (0.9) → small step size → fine quantization
                                           → more bits, high fidelity
    Low attention  (0.1) → large step size → coarse quantization
                                           → fewer bits, acceptable loss
